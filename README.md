## Konfigurasi .env.local

Buat file [`.env.local`] pada root proyek, dengan isi berikut:
```bash
MONGODB_URI
NEXTAUTH_SECRET
NEXTAUTH_URL
```

## Instalasi

```bash
npm install
npm run dev
```
